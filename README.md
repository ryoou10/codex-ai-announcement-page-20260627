# 公開用サイト

このフォルダをNetlifyなどにアップすると、初回モニター募集ページとして使えます。

## 中身

- `index.html`: AI告知ページ制作の商品ページ
- `apply/index.html`: Googleフォーム未作成でも使えるDM申込テンプレート
- `samples/pilates-trial/index.html`: ピラティス体験会の制作サンプル
- `samples/pilates-trial/desktop-preview.png`: サンプルのPCスクショ
- `samples/pilates-trial/mobile-preview.png`: サンプルのスマホスクショ

## 使い方

1. `public-site/` フォルダをNetlifyにアップする
2. 発行されたURLをXプロフィール、固定ポスト、note記事、DMに貼る
3. GoogleフォームURLがない場合は、`apply/index.html` の申込テンプレートをDMで案内する

最初の目的は、完璧なサイト公開ではなく、初回3,000円モニターの申し込み導線を作ること。
